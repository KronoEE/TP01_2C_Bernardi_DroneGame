# TP07Integrador_C1_Bernardi_Dronner

Dronner es un juego de acción en 3D desarrollado en Unity, donde controlás un dron de combate en misiones de alta precisión.

Tu objetivo es eliminar una cantidad específica de enemigos mientras evitás dañar civiles inocentes. Cada decisión cuenta: un disparo mal ejecutado puede costarte puntos y la misión.

Volá, apuntá, dispará y completá el protocolo de ataque con la mayor precisión posible.

## 🕹️ Controles

| Acción | Tecla / Botón |
|--------|----------------|
| Mover dron | `W / A / S / D` |
| Subir | `Espacio` |
| Bajar | `Ctrl Izquierdo` |
| Apuntar | `Mouse` |
| Disparar láser | `Click izquierdo` |
| Disparar cohete | `Q` |
| Pausa | `Esc` |

## ⚔️ Mecánicas Principales

🚁 **Control del dron:** movimiento libre en el espacio con control vertical y horizontal.

🎯 **Sistema de combate:**
- Láser para ataques rápidos y precisos.
- Cohetes para mayor impacto.

👥 **Sistema de civiles y enemigos:**
- Los enemigos deben ser eliminados para completar la misión.
- Los civiles **no deben ser dañados**.

📊 **Sistema de puntuación:**
- Eliminar enemigos suma puntos.
- Dañar civiles resta puntos.

🏁 **Condición de victoria:**
- La misión se completa al eliminar una cantidad específica de enemigos definida por el juego.
